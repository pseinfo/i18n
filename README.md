# Localization

This package provides localization support throughout the [@pseinfo](https://github.com/pseinfo) project.  
**All translations will served from this package.**

## Exports

This package exports static files:

- `src/locales`  
  Contains all translation files in TypeScript format, structured by language code (e.g., `en`, `de`) and namespace.
- `src/config`  
  Contains configuration files for localization, such as supported languages and default language settings.
